# Home Server Roadmap
## Phase 1
### Deploy and Manage Essential Server Services and Networking
- Docker & Docker Compose: Containerization
- SSH
- Static IP & DHCP Configuration
- Firewall (UFW / iptables)
- System Monitoring (Glances / htop / netdata)

## Phase 2
### Networking and Security
- Pi-hole: Network-wide DNS filtering & ad-blocking
- WireGuard / OpenVPN: VPN server
- Nginx/Caddy : Reverse proxy & SSL termination
- Uptime Kuma / Prometheus + Grafana: Service monitoring
- Portainer: Docker management

## Phase 3
### Shared Storage, User Permissions and Collaboration tools
- Nextcloud / ownCloud: Personal cloud, file sharing
- Samba / NFS: Shared network drives
- Syncthing: File synchronization
- Backup Tools (Duplicati / rsnapshot / Borg): Automated backups

## Phase 4
### Media & Entertainment Services
- Jellyfin: Media server
- Calibre Web: E-book library
- Transmission / qBittorrent: Torrent client
- Home Assistant / Node-RED: Smart home automation

## Phase 5 
### 
- Docker Labs (WordPress, Ghost, etc.): Test apps in containers
- Self-hosted Git (Gitea): Track personal projects
- Home Lab Networking (pfSense VM): Router/firewall simulation
- Lightweight Game Servers (Minecraft, Valheim, etc.)
